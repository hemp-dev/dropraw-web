# Support

Use GitHub Issues for bugs, provider requests, docs fixes, and mirror requests after the primary repository is public.

Before opening an issue:

- Run `dropraw doctor`.
- Confirm Python is 3.11 or 3.12.
- Capture the command, provider, source type, retry settings, and relevant error text.
- Remove tokens, credentials, private URLs, and customer data.

For security issues, follow [SECURITY.md](SECURITY.md).

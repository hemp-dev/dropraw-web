# Российские Платформы

## GitFlic

Основной российский public mirror.

```bash
git remote add gitflic git@gitflic.ru:hemp-dev/rawbridge.git
```

## GitVerse

Второй российский public/private mirror.

```bash
git remote add gitverse git@gitverse.ru:hemp-dev/rawbridge.git
```

## Сфера.Код / Платформа Сфера

Enterprise/self-hosted target. Создайте внутренний репозиторий в вашей инсталляции Сфера.Код и добавьте его как remote.

```bash
git remote add sfera git@sfera.example.ru:TEAM/rawbridge.git
```

## РТК-Феникс

Secure supply-chain / dependency repository context. Не описывайте как прямой GitHub replacement без подтвержденного Git-hosting workflow.

## Mos.Hub

Optional public-sector/open-source mirror. Использовать только если registration и mirror workflow доступны и проверены.

## Национальный Репозиторий / РФРИТ

Strategic/government context. Не делать required release target без стабильного публичного процесса.

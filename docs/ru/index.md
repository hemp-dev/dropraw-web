# DropRaw Web

DropRaw Web — open-source CLI и локальный UI для конвертации больших RAW-папок из Dropbox, Google Drive, S3/R2 и локальных папок в WebP/AVIF/JPEG/PNG без огромных ZIP-downloads.

Проверено на реальной Dropbox shared folder с 816 Nikon NEF files. Ошибка `ConnectionError(ProtocolError('Connection aborted.', OSError(22, 'Invalid argument')))` считается временной и recoverable.

Рекомендуемые версии Python: 3.11 и 3.12.

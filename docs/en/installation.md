# Installation

```bash
pip install rawbridge
```

Development install:

```bash
git clone https://github.com/hemp-dev/rawbridge.git
cd rawbridge
python3.12 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
```

Recommended Python versions: 3.11 and 3.12. Python 3.14 is not recommended for this release.

# Product Hunt

## Name

DropRaw Web

## Tagline

Convert large RAW photo folders into optimized web assets without huge ZIP downloads.

## Description

DropRaw Web is an open-source CLI and local UI for photographers, designers, agencies, and developers. It scans Dropbox, Google Drive, S3/R2, and local folders, downloads RAW files one by one, converts them to WebP/AVIF/JPEG/PNG, and resumes safely after failures.

# RawBridge Docs

Language: [English](en/index.md) | [Русский](ru/index.md) | [Español](es/index.md) | [中文](zh/index.md) | [Deutsch](de/index.md) | [Français](fr/index.md)

RawBridge is an open-source CLI and local UI for converting large RAW photo folders from Dropbox, Google Drive, S3/R2, and local folders into optimized WebP/AVIF/JPEG/PNG assets.

It is designed around real-world reliability: provider listing, one-file-at-a-time downloads, retries, `.part` files, failed logs, reports, and resume.

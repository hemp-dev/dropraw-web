# Changelog

All notable changes to this project are documented in this file.

The format follows Keep a Changelog, and this project uses SemVer.

## [0.1.0] - 2026-05-15

### Added

- CLI for RAW to web image conversion.
- Local folder provider.
- Dropbox shared folder provider without ZIP download.
- Google Drive provider.
- S3/R2 provider.
- WebP/JPEG/PNG/AVIF output.
- SQLite manifest.
- Resume support.
- Failed log.
- Retry only failed.
- `.part` download.
- Local UI dashboard.
- Reports JSON/CSV/HTML.
- Docker support.
- International mirror documentation.
- Russian mirror documentation.

### Reliability

- Listing retry for Dropbox shared folders.
- Download retry for large RAW files.
- Exponential backoff.
- Size checks.
- Recoverable transient error handling.

### Documentation

- English docs.
- Russian docs.
- Spanish docs.
- Chinese docs.
- German docs.
- French docs.
- GitHub/GitLab/Bitbucket/Codeberg mirror docs.
- GitFlic/GitVerse Russian mirror docs.
